# 🏥 Health Vault Prototype

## 📌 Overview

Health Vault is a web-based prototype designed to securely store and manage patient health records. It enables interaction between users, doctors, labs, and organizations in a centralized system.

## 🚀 Features

* 👤 User profile & health record storage
* 👨‍⚕️ Doctor access & prescription handling
* 🧪 Lab report uploads
* 🏢 Organization-level access
* 🤖 AI-powered assistant (Gemini API integration)
* 🔐 Firebase authentication & database

## 🛠️ Tech Stack

* HTML, CSS, JavaScript
* Firebase (Auth, Firestore, Storage)
* Chart.js (data visualization)

## 📂 Project Structure

* `index.html` → Main entry point
* `app.js` → Application logic
* `styles.css` → UI styling
* `firebase-config.js` → Firebase setup
* `/images` → Assets

## 🌐 Deployment

Hosted using Vercel
